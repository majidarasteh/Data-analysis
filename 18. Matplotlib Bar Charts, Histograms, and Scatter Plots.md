# **1. Bar Chart – `bar()` and `barh()`**

Bar charts are used to compare categories—such as sales by company or population of countries.

Matplotlib provides two types:

| Function         | Description          |
| ---------------- | -------------------- |
| `plt.bar(x, y)`  | Vertical bar chart   |
| `plt.barh(x, y)` | Horizontal bar chart |

* Use when comparing categorical data.
* Add labels, title, ticks, and legend for clarity.

## **Example: Vertical Bar Chart**

```python
import matplotlib.pyplot as plt

companies = ["A", "B", "C", "D"]
sales = [40, 55, 30, 70]

plt.bar(companies, sales, color='skyblue')

plt.xlabel("Company")
plt.ylabel("Sales")
plt.title("Sales Comparison")
plt.show()
```

## **Example: Multiple Bar Charts**

To show grouped bars:
Call `plt.bar()` multiple times and adjust bar positions.

```python
import numpy as np
import matplotlib.pyplot as plt

labels = ["Spring", "Summer", "Autumn", "Winter"]
x = np.arange(len(labels))

product1 = [20, 34, 30, 35]
product2 = [25, 32, 34, 20]

width = 0.35       # Width of bars and is used to adjust bar positions

plt.bar(x - width/2, product1, width, label="Product 1", color='orange')
plt.bar(x + width/2, product2, width, label="Product 2", color='skyblue')

plt.xlabel("Seasons")
plt.ylabel("Sales")
plt.title("Seasonal Sales Comparison")
plt.xticks(x, labels)
plt.legend()

plt.show()
```

## **Example: Multiple Barh Charts**
* **To show grouped barhs**: Call `plt.barh()` multiple times and adjust bar positions.
* **`plt.yticks(x, labels)`** is used to set the positions and labels of the ticks on the `y-axis` (vertical axis) in Matplotlib.

```python
import numpy as np
import matplotlib.pyplot as plt

labels = ["Spring", "Summer", "Autumn", "Winter"]
x = np.arange(len(labels))

product1 = [20, 34, 30, 35]
product2 = [25, 32, 34, 20]
product3 = [15, 28, 25, 30] 

width = 0.25  # Width of each bar and is used to adjust bar positions

# Plot horizontal bars
plt.barh(x - width, product1, width, label="Product 1", color='orange')
plt.barh(x, product2, width, label="Product 2", color='skyblue')
plt.barh(x + width, product3, width, label="Product 3", color='green')

plt.ylabel("Seasons")
plt.xlabel("Sales")
plt.title("Seasonal Sales Comparison")
plt.yticks(x, labels)
plt.legend()
plt.grid(axis='x', linestyle='--', alpha=0.7)

plt.show()
```

# **2. Histogram – `hist()`**

A **histogram** shows the distribution of numeric data by dividing values into “bins”.

Key points:

* `plt.hist(data)` creates the histogram
* Default bins = 10
* Shows frequency of values inside each bin
* No need for y-values (Matplotlib calculates counts automatically)
* In Matplotlib, the **`alpha` parameter** controls the **transparency (opacity)** of a plot element.

## **Example: Simple Histogram**

```python
import matplotlib.pyplot as plt
import numpy as np

data = np.random.randn(500)

plt.hist(data, bins=20, color='purple', alpha=0.7, edgecolor="blue")

plt.xlabel("Value")
plt.ylabel("Frequency")
plt.title("Normal Distribution Histogram")
plt.show()
```

## **Example: Histogram With Median Line – `axvline()`**

**`plt.axvline()`** useful for statistical insight. Stands for **“axis vertical line”**.
* **Draws a vertical line** across the entire plot at a specified x-coordinate.
* Often used to **highlight thresholds, medians, or specific points in data**.

```python
import numpy as np
import matplotlib.pyplot as plt

data = np.random.randn(1000)
median_val = np.median(data)

plt.hist(data, bins=30, alpha=0.6, color='orange')
plt.axvline(median_val, color='red', linestyle='--', label=f"Median = {median_val:.2f}")

plt.xlabel("Value")
plt.ylabel("Frequency")
plt.title("Histogram with Median")
plt.legend()
plt.grid(True)

plt.show()
```

# **3. Scatter Plot – `scatter()`**

Scatter plots show the relationship between two variables—useful for correlation.

Key parameters:

| Parameter | Meaning            |
| --------- | ------------------ |
| `s`       | marker size        |
| `c`       | color              |
| `marker`  | shape of the point |
| `alpha`   | transparency       |

## **Example 2: Customized Scatter Plot**

Explanation of each parameter in the following code:

| Parameter    | Purpose                                                                                                                |
| ------------ | ---------------------------------------------------------------------------------------------------------------------- |
| `x, y`       | The coordinates of the points to plot on the scatter plot.                                                             |
| `s=sizes`    | **Size of the markers**. Can be a single number (all points same size) or an array/list for different sizes per point. |
| `c=colors`   | **Color of the markers**. Can be a single color (e.g., `'blue'`) or a list/array to give each point a different color. |
| `alpha=0.6`  | **Transparency** of the markers. `0` is fully transparent, `1` is fully opaque.                                        |
| `marker='o'` | Shape of the markers. `'o'` is a circle. Other options: `'*'` (star), `'^'` (triangle), `'s'` (square), etc.           |


```python
import matplotlib.pyplot as plt
import numpy as np

x = np.random.rand(50)
y = np.random.rand(50)
sizes = np.random.randint(50, 300, 50)
colors = np.random.rand(50)

plt.scatter(x, y, s=sizes, c=colors, alpha=0.6, marker='o')

plt.xlabel("X Values")
plt.ylabel("Y Values")
plt.title("Customized Scatter Plot")
plt.show()
```
