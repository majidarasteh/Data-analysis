# **Advanced Visualization Techniques**

# **1. 3D Plots (mplot3d)**

Matplotlib offers a special toolkit called **mplot3d** that allows you to create 3D visualizations. These are useful when exploring relationships between three variables or visualizing surfaces and shapes.

### **Enabling 3D axes**

```python
from mpl_toolkits.mplot3d import Axes3D
import matplotlib.pyplot as plt
import numpy as np

fig = plt.figure(figsize=(8,6))
ax = fig.add_subplot(111, projection='3d')
```

**Explanation**

1. `fig = plt.figure()`:
   * Creates a new figure **(a blank canvas)** where you can draw plots.
   * Think of it as a page in a notebook.
     
2. `ax = fig.add_subplot(111, projection='3d')`
   * Adds a **3D subplot** to the figure.
   * `111` means:
     * 1 row (**number of rows**)
     * 1 column (**number of columns**)
     * 1 → first (and only) subplot in the grid. Position of the subplot (counted left to right, top to bottom, starting from 1)
    * **Other examples:**
      * `fig.add_subplot(221)` → 2 rows × 2 columns, subplot 1 (top-left)
      * `fig.add_subplot(222)` → 2 rows × 2 columns, subplot 2 (top-right)
      * `fig.add_subplot(223)` → 2 rows × 2 columns, subplot 3 (bottom-left)
      * `fig.add_subplot(224)` → 2 rows × 2 columns, subplot 4 (bottom-right)
        
    * **Visual layout for $2×2$ grid:**

      ```
      +---------+---------+
      |   221   |   222   |
      +---------+---------+
      |   223   |   224   |
      +---------+---------+
      ```

    * `projection='3d'` turns this subplot into a 3D axes, allowing 3D plotting.

3. `ax.plot(x, y, z, linewidth=2)`
   * Plots a **3D** line using the x, y, and z coordinates.
   * linewidth=2 makes the line a bit thicker than the default.

### **3D Line Plot**

```python
from mpl_toolkits.mplot3d import Axes3D
import matplotlib.pyplot as plt
import numpy as np

# Sample data
t = np.linspace(0, 10, 100)
x = np.sin(t)
y = np.cos(t)
z = t

# Create 3D figure
fig = plt.figure()
ax = fig.add_subplot(111, projection='3d')

# Plot 3D line
ax.plot(x, y, z, linewidth=2)

# Add title and axis labels
ax.set_title("3D Line Plot")
ax.set_xlabel("X-axis (sin(t))")
ax.set_ylabel("Y-axis (cos(t))")
ax.set_zlabel("Z-axis (t)")

plt.show()
```

### **3D Scatter Plot**

```python
from mpl_toolkits.mplot3d import Axes3D
import matplotlib.pyplot as plt
import numpy as np

# Sample data
x = np.random.random(100)
y = np.random.random(100)
z = np.random.random(100)

# Create figure and 3D axes
fig = plt.figure()
ax = fig.add_subplot(111, projection='3d')

# Color points based on z value
scatter = ax.scatter(x, y, z, c=z, cmap='coolwarm', s=50, alpha=0.8)

# Add title and axis labels
ax.set_title("3D Scatter Plot with Colors")
ax.set_xlabel("X-axis")
ax.set_ylabel("Y-axis")
ax.set_zlabel("Z-axis")

plt.show()
```

### **3D bar chart**

`np.meshgrid` **takes two 1D arrays** (usually representing the `x-axis` and `y-axis` points) and **turns them into 2D coordinate grids**. This is commonly used for:
* **3D** surface plots (`ax.plot_surface`)
* Contour plots (`plt.contour`)
* Any situation where you need every **combination of x and y coordinates**
  
```python
import matplotlib.pyplot as plt
import numpy as np

# Sample data
x = np.arange(5)
y = np.arange(4)
xpos, ypos = np.meshgrid(x, y)
xpos = xpos.flatten()
ypos = ypos.flatten()
zpos = np.zeros_like(xpos)  # all bars start from z=0

dx = dy = 0.5  # width of bars
dz = xpos**2-3*ypos  # height of bars

# Create figure and 3D axes
fig = plt.figure()
ax = fig.add_subplot(111, projection='3d')

# Create bars
ax.bar3d(xpos, ypos, zpos, dx, dy, dz, color='blue', alpha=0.8)

# Labels
ax.set_xlabel("X-axis")
ax.set_ylabel("Y-axis")
ax.set_zlabel("Z-axis")
ax.set_title("Simple 3D Bar Chart")

plt.show()
```

## **2. 3D Surface Plot**

```python
fig = plt.figure()
ax = fig.add_subplot(111, projection='3d')

X = np.linspace(-5, 5, 50)
Y = np.linspace(-5, 5, 50)
X, Y = np.meshgrid(X, Y)
Z = np.sin(np.sqrt(X**2 + Y**2))

ax.plot_surface(X, Y, Z, cmap='viridis')

# Add title and axis labels
ax.set_title("3D Surface Plot")
ax.set_xlabel("X-axis")
ax.set_ylabel("Y-axis")
ax.set_zlabel("Z-axis")

plt.show()
```
