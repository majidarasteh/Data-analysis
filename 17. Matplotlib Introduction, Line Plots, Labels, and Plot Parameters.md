# Matplotlib Tutorial

# **1. Understanding Matplotlib**

**Matplotlib** is a powerful Python library used for data visualization.
It provides extensive tools to create:

* 2D charts (line, bar, histogram, scatter, pie…)
* 3D visualizations
* Animations
* Heatmaps
* Subplots and combined figures

The most commonly used module is:

```python
import matplotlib.pyplot as plt
```

---

# **2. Line Plot – `plot()` Function**

A **line plot** is one of the most basic types of visualizations.
You create a simple line chart using:

```python
plt.plot()
```

To display the figure, we must always call:

```python
plt.show()
```

### **Example: A Simple Line Chart**

```python
import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y = [2, 3, 5, 6, 7]

plt.plot(x, y)
plt.show()
```

# **3. Adding Title and Axis Labels**

To add descriptive information:

| Function       | Description       |
| -------------- | ----------------- |
| `plt.xlabel()` | Adds X-axis label |
| `plt.ylabel()` | Adds Y-axis label |
| `plt.title()`  | Adds chart title  |


### **Example: Adding title and labels**

```python
import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [2, 4, 3, 5]

plt.plot(x, y)

plt.xlabel("X Values")
plt.ylabel("Y Values")
plt.title("Simple Line Plot")

plt.show()
```

# **4. Plot Parameters (marker, linestyle, color, grid, label)**

Matplotlib provides custom styling options:

| Parameter        | Purpose                      | Example                     |
| ---------------- | ---------------------------- | --------------------------- |
| `marker`         | highlights data points       | `'o'`, `'s'`, `'*'`         |
| `linestyle`      | style of the connecting line | `'--'`, `'-.'`, `':'`       |
| `color`          | color of the line            | `'g'` (green), `'r'`, `'b'` |
| `label`          | name for legend              | `'My Line'`                 |
| `plt.grid(True)` | shows a grid                 | —                           |



### **Example: Using parameters in `plot()`**

```python
import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [2, 3, 2, 5]

plt.plot(x, y,
         marker='o',
         linestyle='--',
         color='g',
         label="Sample Line")

plt.title("Plot with Custom Parameters")
plt.xlabel("X Axis")
plt.ylabel("Y Axis")
plt.grid(True)
plt.legend()

plt.show()
```

# **5. Plotting Multiple Lines + Legend**

To draw multiple lines on the same figure:

* Call `plt.plot()` multiple times
* Give each line a different color and label
* **Add `plt.legend()` at the end**

### **Example: Multiple Line Charts**

```python
import matplotlib.pyplot as plt

x = ["A", "B", "C", "D"]

y1 = [1, 4, 2, 5]
y2 = [2, 2, 3, 4]

plt.plot(x, y1, color='g', label='Line 1')
plt.plot(x, y2, color='r', label='Line 2')

plt.xlabel("X Axis")
plt.ylabel("Y Axis")
plt.title("Multiple Line Plot")
plt.legend()
plt.grid()

plt.show()
```

# **6. Saving a Figure – `savefig()`**

Instead of only displaying a chart, you can save it:

```python
plt.savefig("my_figure.png")
```

### **Example: Saving a Chart**

```python
import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [3, 5, 2, 6]

plt.plot(x, y)
plt.title("Saved Chart Example")

plt.savefig("saved_chart.png")
plt.show()
```

This saves the image in your working directory.
