# **1. Pie Chart – `pie()`**

Pie charts show how different categories contribute to a whole.
They work best when the number of categories is **6 or fewer**.

Key parameters:

| Parameter | Description                        |
| --------- | ---------------------------------- |
| `labels`  | category names                     |
| `colors`  | colors of slices                   |
| `autopct` | percentage format (e.g., "%.2f%%") |
| `explode` | separate one slice from the others |
| `shadow`  | adds a shadow effect               |


## **Example 1: Basic Pie Chart**

```python
import matplotlib.pyplot as plt

sales = [25, 30, 20, 15, 10]
labels = ["Market A", "Market B", "Market C", "Market D", "Market E"]

plt.pie(sales, labels=labels, autopct="%0.1f%%")

plt.title("Market Share")
plt.show()
```

## **Example 2: Pie Chart with Explode, Shadow, Custom Colors**

**Simple explanation:**
* `explode` → highlight a slice.
* `shadow` → make the pie look 3D.
* `colors` → pick your favorite colors for slices.
* `autopct` → show percentages on the chart.

```python
import matplotlib.pyplot as plt

values = [40, 25, 20, 15]
labels = ["Product A", "Product B", "Product C", "Product D"]
explode = [0.1, 0, 0, 0]        # Highlight the first slice
colors = ["gold", "lightgreen", "lightblue", "pink"]

plt.pie(values,
        labels=labels,
        explode=explode,
        colors=colors,
        autopct="%1.2f%%",
        shadow=True)

plt.title("Annual Sales Distribution")
plt.show()
```

# **2. Boxplot – `boxplot()`**

Boxplots summarize the distribution of data using **quartiles**, **median**, and **outliers**.

Common use cases:

* Detecting outliers
* Comparing distributions
* Visualizing spread of values

Key parameters:

| Parameter           | Description           |
| ------------------- | --------------------- |
| `vert=False`        | horizontal boxplot    |
| `patch_artist=True` | fill boxes with color |
| `boxprops`          | style of box          |
| `flierprops`        | style of outliers     |

---

## **Example 1: Basic Boxplot**

```python
import matplotlib.pyplot as plt
import numpy as np

data = np.random.normal(50, 10, 200) # Normal distribution: mean: 50, standard deviation = 10, Number of random values=200

plt.boxplot(data)
plt.title("Basic Boxplot")
plt.ylabel("Value")
plt.show()
```

## **Example 2: Customized Boxplot**

```python
import matplotlib.pyplot as plt
import numpy as np

data = [np.random.normal(60, 8, 200),
        np.random.normal(50, 10, 200),
        np.random.normal(55, 6, 200)]

plt.boxplot(data,
            patch_artist=True,
            boxprops=dict(facecolor="lightblue"),
            flierprops=dict(marker='o', markerfacecolor='red')) 

plt.title("Customized Boxplot")
plt.xlabel("Groups")
plt.ylabel("Values")
plt.show()
```

# **3. Heatmap – `imshow()`**

`imshow()` is used for:

* Displaying images (arrays)
* Visualizing 2D data matrices
* Creating heatmaps (correlation matrices, etc.)

| Parameter        | Purpose                                                                             |
| ---------------- | ----------------------------------------------------------------------------------- |
| `data`           | 2D array (matrix) to visualize                                                      |
| `cmap`           | Colormap used to map values to colors (e.g., `'viridis'`, `'coolwarm'`, `'plasma'`) |
| `plt.colorbar()` | Adds a color scale to interpret values                                              |

**What Do Correlation Values Mean?**

Correlation coefficient values range from $–1$ to $+1$:

| Correlation Value | Meaning                       | Example                                   |
| ----------------- | ----------------------------- | ----------------------------------------- |
| **+1**            | Perfect positive relationship | As study time increases, grades increase  |
| **0**             | No relationship               | Shoe size vs salary                       |
| **–1**            | Perfect negative relationship | As speed increases, travel time decreases |


**Why Use a Heatmap for Correlation?**
* A correlation matrix contains many numbers.
  * A heatmap makes **patterns easier to see** because:
  * **Strong positive correlations** → warm colors (e.g., yellow, bright green depending on `cmap`)
  * **Strong negative correlations** → cool colors (e.g., dark purple, blue, red depending on cmap)
  * **Near zero correlation** → middle colors (neutral tones)

**Points**
* A correlation matrix shows how variables relate (from $–1$ to $+1$).
* A heatmap turns numbers into colors for easy interpretation.
* Strong correlations → intense colors
* Weak correlations → neutral colors
 
## **Example**

```python
import numpy as np
import matplotlib.pyplot as plt

# Simulate a correlation matrix
corr = np.array([
    [1.00, 0.82, 0.10, 0.05],
    [0.82, 1.00, 0.08, 0.03],
    [0.10, 0.08, 1.00, 0.60],
    [0.05, 0.03, 0.60, 1.00],
])

plt.imshow(corr, cmap='coolwarm')
plt.colorbar()
plt.title("Correlation Matrix Heatmap")
plt.show()
```

## **Example: Heatmap for Correlation Matrix**
In `np.corrcoef(data, rowvar=False)`, the argument `rowvar=False` tells NumPy how your data is organized before computing the correlation matrix.

**`rowvar=False` means:**
* Each column in your data represents a variable (feature).
* Each row represents an observation (sample).
* NumPy should compute correlations between columns, not between rows.

**Why Do We Need This?**
* By default, NumPy assumes each row is a variable (`rowvar=True`).
* But most datasets are structured like this:
  
  | Sample | Height | Weight | Age |
  | ------ | ------ | ------ | --- |
  | 1      | 170    | 65     | 25  |
  | 2      | 180    | 80     | 30  |
  | 3      | 160    | 55     | 22  |

* **Here**:
  * **Columns** → Height, Weight, Age → variables
  * **Rows** → samples
  
**What Happens If You Don’t Use rowvar=False?**
* `rowvar=True` (default) treats each row as a variable.
* You would get a wrong correlation matrix (variables swapped with samples).

```python
import matplotlib.pyplot as plt
import numpy as np

data = np.random.randn(100, 4) # 100 is the total number of rows (samples), and 4 is the total number of features
corr_matrix = np.corrcoef(data, rowvar=False)

plt.imshow(corr_matrix, cmap='coolwarm')
plt.colorbar()
plt.title("Correlation Heatmap")
plt.show()
```

# **4. Displaying an Image – `imshow()`**

1. `mpimg.imread()` loads an image file:
   * It reads the picture from your computer.
   * It converts the image **into a NumPy array** so **that Matplotlib can display it**.
   * `img` now stores the pixel data of the image.
   * Now you can display it using `plt.imshow(img)` and `plt.axis("off")`:

If you have an actual image for example in `"C:\flower.jpg"`

```python
import matplotlib.pyplot as plt
import matplotlib.image as mpimg

img = mpimg.imread(r"C:\flower.jpg")
plt.imshow(img)
plt.axis("off")
plt.show()
```
