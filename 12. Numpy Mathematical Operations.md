# **7. Mathematical Operations (NumPy)**

NumPy provides efficient tools for **element-wise operations**, **matrix operations**, **aggregation**, and more. This tutorial covers all essential operations for **1-D vectors** and **2-D matrices**, including **linear algebra examples**.


## **1.1 Element-wise Arithmetic Operations**

Element-wise operations perform arithmetic **on corresponding elements** of arrays of the same shape.

```python
import numpy as np

# 1-D arrays
a1 = np.array([1, 2, 3])
a2 = np.array([4, 5, 6])

print("1-D Array Addition:")
print(a1 + a2)
print(np.add(a1, a2))

print("\n1-D Array Subtraction:")
print(a2 - a1)
print(np.subtract(a2, a1))

print("\n1-D Array Multiplication:")
print(a1 * a2)
print(np.multiply(a1, a2))

print("\n1-D Array Division:")
print(a2 / a1)
print(np.divide(a2, a1))
print(a2 // a1)
print(np.floor_divide(a2, a1))

print("\n1-D Array Remainder:")
print(a2 % a1)
print(np.remainder(a2, a1))
```

**Output:**

```
1-D Array Addition:
[5 7 9]
[5 7 9]

1-D Array Subtraction:
[3 3 3]
[3 3 3]

1-D Array Multiplication:
[ 4 10 18]
[ 4 10 18]

1-D Array Division:
[4.  2.5 2. ]
[4.  2.5 2. ]
[4 2 2]
[4 2 2]

1-D Array Remainder:
[0 1 0]
[0 1 0]
```

**2-D Arrays**

```python
# 2-D arrays
mat1 = np.array([[1, 2, 3],
                 [4, 5, 6]])
mat2 = np.array([[7, 8, 9],

print("\n2-D Array Addition:")
print(mat1 + mat2)

print("\n2-D Array Division:")

print(np.floor_divide(mat2, mat1))

print("\n2-D Array Remainder:")
```

## **1.3 Matrix Transpose**

```python
mat = np.array([[1, 2, 3],
                [4, 5, 6]])

print("Original 2-D matrix:")
print(mat)

mat_T = mat.T
print("\nTransposed 2-D matrix:")
print(mat_T)
print("Shape:", mat_T.shape)
```

**Output:**

```
Original 2-D matrix:
[[1 2 3]
 [4 5 6]]

Transposed 2-D matrix:
[[1 4]
 [2 5]
 [3 6]]
Shape: (3, 2)
```

## **1.4 Dot Product (Matrix Multiplication)**

**1-D Vectors**

```python
dot_vec = np.dot(a1, a2)
print("Dot Product 1-D:", dot_vec)
```

**Output:**

```
Dot Product 1-D: 32
```

**2-D Matrices**

```python
A = np.array([[1, 2, 3],
              [4, 5, 6]])
B = np.array([[7, 8],
              [9, 10],
              [11, 12]])

C = np.dot(A, B)
print("Dot Product 2-D (A·B):")
print(C)
```

**Output:**

```
Dot Product 2-D (A·B):
[[ 58  64]
 [139 154]]
```

## **1.7 Append and Delete**

```python
# 1-D
arr1 = np.array([1,2,3])
arr2 = np.append(arr1, [4,5])
print("Append 1-D:", arr2)
arr3 = np.delete(arr2, 1)
print("Delete index 1:", arr3)

# 2-D
mat_appended = np.append(mat1, [[7,8,9]], axis=0)
print("Append new row 2-D:\n", mat_appended)
mat_deleted = np.delete(mat_appended, 1, axis=0)
print("Delete second row 2-D:\n", mat_deleted)
```

**Output:**

```
Append 1-D: [1 2 3 4 5]
Delete index 1: [1 3 4 5]

Append new row 2-D:
[[1 2 3]
 [4 5 6]
 [7 8 9]]
Delete second row 2-D:
[[1 2 3]
 [7 8 9]]
```

## **1.8 Aggregation Functions**

```python
# 1-D
print("1-D Sum:", np.sum(a1))
print("1-D Mean:", np.mean(a1))

# 2-D
print("\n2-D Sum:", np.sum(mat1))
print("Sum along columns:", np.sum(mat1, axis=0))
print("Sum along rows:", np.sum(mat1, axis=1))
print("Mean 2-D:", np.mean(mat1))
print("Max:", np.max(mat1))
print("Min:", np.min(mat1))
print("Std Dev:", np.std(mat1))
print("Variance:", np.var(mat1))
```

**Output:**

```
1-D Sum: 6
1-D Mean: 2.0

2-D Sum: 21
Sum along columns: [5 7 9]
Sum along rows: [ 6 15]
Mean 2-D: 3.5
Max: 6
Min: 1
Std Dev: 1.707825127659933
Variance: 2.9166666666666665
```

| Multiplication     | `*`, `np.multiply()`    | `a1 * a2`             | `mat1 * mat2`                       | Element-wise multiplication  |
| Division           | `/`, `//`               | `a2 / a1`             | `mat2 / mat1`                       | True and floor division      |
| Modulus            | `%`                     | `a2 % a1`             | `mat2 % mat1`                       | Element-wise remainder       |
| Power              | `**`, `np.square()`     | `a1 ** 2`             | `np.square(mat1)`                   | Raise elements to a power    |
| Transpose          | `.T`                    | N/A                   | `mat.T`                             | Swap rows & columns          |
| Dot Product        | `np.dot()`              | `np.dot(a1,a2)`       | `np.dot(A,B)`                       | Vector/matrix multiplication |
| Exponential        | `np.exp()`              | `np.exp(a1)`          | `np.exp(mat)`                       | e^x element-wise             |
| Logarithm          | `np.log()`              | `np.log(a2)`          | `np.log(mat)`                       | Natural log element-wise     |
| Broadcasting       | Automatic               | `a1*2`                | `mat*10`                            | Scalar/array broadcasting    |
| Append             | `np.append()`           | `np.append(a1,[4,5])` | `np.append(mat, [[7,8,9]], axis=0)` | Add elements along axis      |
| Delete             | `np.delete()`           | `np.delete(arr,1)`    | `np.delete(mat,1,axis=0)`           | Remove elements along axis   |
| Sum                | `np.sum()`              | `np.sum(a1)`          | `np.sum(mat,axis=0)`                | Sum of elements              |
| Mean               | `np.mean()`             | `np.mean(a1)`         | `np.mean(mat)`                      | Average value                |
