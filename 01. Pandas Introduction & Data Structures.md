Pandas is an open-source Python library providing fast, flexible, and easy-to-use data structures and data analysis tools.

**Advantages of Pandas:**

* Easy to use
* **Integrates** well with NumPy, Matplotlib, Scikit-Learn
* Ideal for **preparing raw data** for machine learning
* Can read/write many data formats (**CSV, Excel, TSV, etc.**)

# **3. Pandas Data Structures**
Pandas has two main data structures: **Series** (1D) and **DataFrame** (2D).

### **3.1 Series (1-Dimensional)**

A labeled one-dimensional array.

* Mutable data values
* Fixed size
* Index starts from 0

### **Code Example 1**

```python
import pandas as pd
s = pd.Series([10, 20, 30])
print(s)
```

**Output**

```
0    10
1    20
2    30
dtype: int64
```

### **Code Example 2**

```python
import pandas as pd

# Create a simple Series with string values
names = pd.Series(["Alice", "Bob", "Charlie"])

print(names)
```

**Output**

```
0       Alice
1         Bob
2    Charlie
dtype: object
```

### **Code Example 3**

```python
import pandas as pd

# Create a simple Series with mixed data
names = pd.Series(["Alice", "Bob", "Charlie", 1, 2, 3])

print(names)
```

**Output**

```
0      Alice
1        Bob
2    Charlie
3          1
4          2
5          3
dtype: object
```

---

### **3.2 DataFrame (2-Dimensional)**

A **table-like structure** consisting of multiple Series objects (columns). **Similar to Excel** spreadsheets.

### **Code Example 1**

```python
df = pd.DataFrame({
    'Name': ['A', 'B', 'C'],
    'Age': [20, 25, 30]
})
print(df)
```

**Output**

```
  Name  Age
0    A   20
1    B   25
2    C   30
```

You can combine multiple Series into one DataFrame.

### **Code Example 2**

```python
s1 = pd.Series(['BMW', 'Toyota', 'Honda'])
s2 = pd.Series([2019, 2020, 2021])

df = pd.DataFrame({'Car': s1, 'Year': s2})
print(df)
```

**Output**

```
      Car  Year
0     BMW  2019
1  Toyota  2020
2   Honda  2021
```

### 3.3 Creating a Series with Custom Index Values

By default, Pandas assigns integer indexes starting from 0. To create a Series with **custom indexes**, **pass two lists**: one for data and one for indexes. **Indexes can be duplicated.**

```python
import pandas as pd
animals = pd.Series(
    ['cat', 'dog', 'bird', 'lion', 'tiger'],
    index=[0, 3, 9, 8, 3]
)
print(animals)
```

**Output**

```
0      cat
3      dog
9     bird
8     lion
3    tiger
dtype: object
```

### 3.4. Multi-Indexing (Hierarchical Indexing) — `set_index()`
Used to define **multi-level indexes**.

**Example**

```python
data = {
    'department': ['medical', 'medical', 'engineering', 'engineering', 'engineering'],
    'Employee' : ['Sina', 'Mina', 'Ali', 'Reza', 'Sara'],
    'Grade': [15, 19, 14, 18, 17]
}

new_df = pd.DataFrame(data)
new_df = new_df.set_index(['department', 'Employee'])
```

**Output**

```
                      Grade
department  Employee       
medical     Sina         15
            Mina         19
engineering Ali          14
            Reza         18
            Sara         17
```

### 3.5.1 Accessing Custom Index Values – `loc` (Series)
Use `.loc[]` to access elements using their index labels.

```python
print("loc[3]:\n",animals.loc[3])   # returns two values (because index=3 appears twice)
print("\nloc[9]:",animals.loc[9])
```

**Output**

```
loc[3]:
3      dog
3    tiger
dtype: object

loc[9]: bird
```

### 3.5.2. Accessing Custom Index Values – `loc` (DataFrame)

`loc` works for DataFrames as well.

```python
df.loc[2]   # returns the row with index label = 2
```

**Output**

```
Name     C
Age     30
Name: 2, dtype: object
```

### 3.5.3. Accessing Custom Index Values – `iloc` (Series & DataFrame)
`iloc` is used to access data by integer position, not by index label. It always works **with 0-based positions**, even if **your index contains custom or duplicated** labels.

* `iloc` ignores index labels completely.
* It selects rows/values **by position only**.
* Useful when your index is not clean, not unique, or not sorted.

#### Example (Series with Custom Index)

```python
animals = pd.Series(
    ['cat', 'dog', 'bird', 'lion', 'tiger'],
    index=[0, 3, 9, 8, 3]
)

print(animals)
```

**Series index looks like:**

| Position | Index | Value |
| -------- | ----- | ----- |
| 0        | 0     | cat   |
| 1        | 3     | dog   |
| 2        | 9     | bird  |
| 3        | 8     | lion  |
| 4        | 3     | tiger |


**Using iloc**

```python
print(animals.iloc[0])   # first element → 'cat'
print(animals.iloc[3])   # fourth element → 'lion'
print(animals.iloc[-1])  # last element → 'tiger'
```

**output**

```
cat
lion
tiger
```

Even though the index contains values like 3 and 9, `iloc` does not use these. It **only looks at positions** 0, 1, 2, 3, 4.

#### Example with DataFrame

```python
df = pd.DataFrame({
    'A': [10, 20, 30],
    'B': [40, 50, 60]
}, index=[5, 7, 9])

print(df)
```

**Output**

```
    A   B
5  10  40
7  20  50
9  30  60
```

**Using `iloc`**

```python
print("--------\n",df.iloc[0])    # row in position 0  (index label = 5)
print("--------\n",df.iloc[2])    # row in position 2  (index label = 9)
print("--------\n",df.iloc[:, 1]) # second column (by position)
print("--------\n",df.iloc[1, 0]) # element at row pos 1, column pos 0 → 20
print("--------\n",df.iloc[0:2])  # rows at positions 0 and 1
```

**Output**

```
--------
 A    10
B    40
Name: 5, dtype: int64
--------
 A    30
B    60
Name: 9, dtype: int64
--------
 5    40
7    50
9    60
Name: B, dtype: int64
--------
 20
--------
     A   B
5  10  40
7  20  50
```

### 3.5.4. MultiIndex Selection Examples

#### **Example: Select the grade of employee "Sina" in the medical department:**

```python
new_df.loc[('medical', 'Sina')]
```

**Output**

```
Grade    15
Name: (medical, Sina), dtype: int64
```

#### Example: Select All Employees in a Department

```python
new_df.loc[('medical', 'Sina')]
```

**Output**

```
	Grade
Employee	
Ali	14
Reza	18
Sara	17
```

#### Example: Select Multiple Index Values

```python
new_df.loc[[('medical', 'Sina'), ('engineering', 'Ali')]]
```

**Output**

```
		Grade
department	Employee	
medical	Sina	15
engineering	Ali	14

```


### 3.5.5. When Do `loc` and `iloc` Give the Same Output?
If you are using the **default index (0…n)**, then:

```python
df.loc[3]   # label 3
df.iloc[3]  # row in position 3
```

These will return the same result.
