# **1. Sorting & Combining Arrays (NumPy)**

NumPy provides efficient tools for **sorting arrays**, **finding indices**, and **combining arrays** both row-wise and column-wise. These are essential for data analysis, preprocessing, and matrix manipulations.


## **1.1 Sorting Arrays**

Sorting arranges elements in **ascending or descending order**.

### **1-D Array Sorting**

```python
import numpy as np

arr1D = np.array([5, 2, 9, 1, 7])
print("Original 1-D array:", arr1D)

# Ascending sort
sorted_arr = np.sort(arr1D)
print("Sorted 1-D array (ascending):", sorted_arr)

# Descending sort
desc_arr = np.sort(arr1D)[::-1]
print("Sorted 1-D array (descending):", desc_arr)
```

**Output:**

```
Original 1-D array: [5 2 9 1 7]
Sorted 1-D array (ascending): [1 2 5 7 9]
Sorted 1-D array (descending): [9 7 5 2 1]
```

### **2-D Array Sorting**

Sorting can be done along **rows (axis=1)** or **columns (axis=0)**. Default sorting mechanism is row-wise: **rows (axis=1)**

```python
arr2D = np.array([[3, 1, 8],
                  [9, 5, 7],
                  [0, 6, 2]])
print("Original 2-D array:\n", arr2D)

# Sort along rows (axis=1)
row_sorted = np.sort(arr2D, axis=1)
print("\nRow-wise sorted:\n", row_sorted)

# Sort along columns (axis=0)
col_sorted = np.sort(arr2D, axis=0)
print("\nColumn-wise sorted:\n", col_sorted)
```

**Output:**

```
Original 2-D array:
 [[3 1 8]
  [9 5 7]
  [0 6 2]]

Row-wise sorted:
 [[1 3 8]
  [5 7 9]
  [0 2 6]]

Column-wise sorted:
 [[0 1 2]
  [3 5 7]
  [9 6 8]]
```

## **1.2 Argsort — Indices of Sorted Elements**

`argsort()` returns **indices that would sort an array**.

```python
arr = np.array([50, 20, 30, 10])
indices = np.argsort(arr)
print("Original array:", arr)
print("Indices that sort the array:", indices)

# Use indices to sort array
sorted_arr = arr[indices]
print("Sorted array using indices:", sorted_arr)
```

**Output:**

```
Original array: [50 20 30 10]
Indices that sort the array: [3 1 2 0]
Sorted array using indices: [10 20 30 50]
```

### **2-D Array Argsort**

```python
arr2D = np.array([[8, 3, 5],
                  [1, 7, 2]])
row_argsort = np.argsort(arr2D, axis=1)
col_argsort = np.argsort(arr2D, axis=0)
print("Row-wise argsort:\n", row_argsort)
print("Column-wise argsort:\n", col_argsort)
```

**Output:**

```
Row-wise argsort:
[[1 2 0]
 [0 2 1]]

Column-wise argsort:
[[1 0 1]
 [0 1 0]]
```

## **1.3 Combining Arrays**

NumPy provides several functions to **combine arrays** along different axes.

### **1-D Arrays**

```python
a = np.array([1, 2, 3])
b = np.array([4, 5, 6])

# Concatenate
concat = np.concatenate([a, b])
print("Concatenated 1-D arrays:", concat)

# Horizontal stack (same as concatenate for 1-D)
hstack_arr = np.hstack([a, b])
print("Horizontally stacked 1-D arrays:", hstack_arr)
```

**Output:**

```
Concatenated 1-D arrays: [1 2 3 4 5 6]
Horizontally stacked 1-D arrays: [1 2 3 4 5 6]
```

---

### **2-D Arrays**

```python
mat1 = np.array([[1, 2],
                 [3, 4]])
mat2 = np.array([[5, 6],
                 [7, 8]])

# Concatenate along rows (axis=0)
concat_rows = np.concatenate([mat1, mat2], axis=0)
print("Concatenated along rows:\n", concat_rows)

# Concatenate along columns (axis=1)
concat_cols = np.concatenate([mat1, mat2], axis=1)
print("\nConcatenated along columns:\n", concat_cols)

# Horizontal stack (same as axis=1)
hstack_mat = np.hstack([mat1, mat2])
print("\nHorizontally stacked:\n", hstack_mat)

# Vertical stack (same as axis=0)
vstack_mat = np.vstack([mat1, mat2])
print("\nVertically stacked:\n", vstack_mat)
```

**Output:**

```
Concatenated along rows:
[[1 2]
 [3 4]
 [5 6]
 [7 8]]

Concatenated along columns:
[[1 2 5 6]
 [3 4 7 8]]

Horizontally stacked:
[[1 2 5 6]
 [3 4 7 8]]

Vertically stacked:
[[1 2]
 [3 4]
 [5 6]
 [7 8]]
```


## **1.4 Splitting Arrays**

Use `np.split()` to **divide arrays** into equal parts.

### **1-D Array Split**

```python
arr = np.array([0, 1, 2, 3, 4, 5])
split_arr = np.split(arr, 3)
print("1-D array split into 3 parts:", split_arr)
```

**Output:**

```
1-D array split into 3 parts: [array([0, 1]), array([2, 3]), array([4, 5])]
```

### **2-D Array Split**

```python
arr2D = np.array([[1, 2, 3, 4],
                  [5, 6, 7, 8]])

# Split into 2 along columns
split_cols = np.split(arr2D, 2, axis=1)
print("\n2-D array split along columns:\n", split_cols)

# Split into 2 along rows
split_cols = np.split(arr2D, 2, axis=0)
print("\n2-D array split along rows:\n", split_cols)
```

**Output:**

```
2-D array split along columns:
 [array([[1, 2],
       [5, 6]]), array([[3, 4],
       [7, 8]])]

2-D array split along rows:
 [array([[1, 2, 3, 4]]), array([[5, 6, 7, 8]])]
```

## **1.5 Summary Table — Sorting & Combining Arrays**

| **Operation**    | **Function / Method**        | **Description**                                | **Example**                     |
| ---------------- | ---------------------------- | ---------------------------------------------- | ------------------------------- |
| Sort 1-D Array   | `np.sort()`                  | Sort elements in ascending or descending order | `np.sort(arr1D)`                |
| Sort 2-D Array   | `np.sort(arr, axis=0/1)`     | Sort along rows or columns                     | `np.sort(arr2D, axis=1)`        |
| Argsort          | `np.argsort()`               | Indices that would sort the array              | `np.argsort(arr)`               |
| Concatenate      | `np.concatenate()`           | Combine arrays along specified axis            | `np.concatenate([a,b], axis=0)` |
| Horizontal Stack | `np.hstack()`                | Stack arrays horizontally (columns)            | `np.hstack([mat1, mat2])`       |
| Vertical Stack   | `np.vstack()`                | Stack arrays vertically (rows)                 | `np.vstack([mat1, mat2])`       |
| Split 1-D Array  | `np.split(arr, n)`           | Split array into n equal parts                 | `np.split(arr, 3)`              |
| Split 2-D Array  | `np.split(arr, n, axis=0/1)` | Split array along specified axis               | `np.split(arr2D, 2, axis=1)`    |
