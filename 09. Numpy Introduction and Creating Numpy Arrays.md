# **1. Introduction to NumPy**


## **1.1 What is NumPy?**

NumPy (short for **Numerical Python**) is a popular Python library used for numerical computing.
It provides:

* Support for **large, multi-dimensional arrays and matrices**
* A large collection of **fast mathematical functions**
* Operations written in **C**, making NumPy extremely fast
* The foundation for data science and machine learning in Python

## **1.2 Installing NumPy**

NumPy can be installed using `pip`, Python’s package manager.

```bash
pip install numpy
```

In Anaconda you can install using:

```bash
conda install numpy
```


To verify installation:

```bash
pip show numpy
```

## **1.3 Why NumPy?**

Python lists are simple but limited:

* They handle only **1-D arrays**
* They are **slow**, because operations require Python loops
* NumPy arrays are **homogeneous** and stored more efficiently
* Most machine learning libraries (Pandas, TensorFlow, PyTorch, Scikit-Learn) depend on NumPy

NumPy arrays allow:

* **Vectorized operations (no loops)**: performing operations on whole NumPy arrays at once, instead of element-by-element loops.

  ```bash
  a1              # a Numpy array
  result = a1*2   # Instead of using for loops, easily multiply each element of a1 by 2
  ```
  
* **Broadcasting**: is a powerful mechanism that allows arrays of different shapes to be used together in arithmetic operations without explicitly writing loops.
  
  ```bash
  array + scalar
  array + vector
  matrix + vector
  ```
  
* **Fast computation** (written in C)
* **Easy integration with data science tools**: Examples:
  * **Pandas uses NumPy** arrays to store data in DataFrames.
  * **Scikit-learn uses NumPy** arrays for machine learning models.
  * **Matplotlib uses NumPy** arrays for plotting.
  * **TensorFlow and PyTorch convert NumPy** arrays to tensors easily.


## **1.4 Import NumPy**

Before using NumPy, import it with a standard alias:

```python
import numpy as np
```

# **2. Creating Arrays**


## **2.1 Creating a 1-D Array (Vector)**

### Example:

```python
import numpy as np

a1 = np.array([1, 2, 3])
print(a1)
```

### Output:

```
[1 2 3]
```

---

## **2.2 Converting a Python List to a NumPy Array**

```python
import numpy as np

myPythonList = [1, 9, 8, 3]
numpy_array_from_list = np.array(myPythonList)
print(numpy_array_from_list)
```

### Output:

```
[1 9 8 3]
```

## **2.3 The `shape` Attribute**

Shows array dimensions.

```python
a = np.array([[1,2,3],
              [4,5,6]])
print(a.shape)
```

### Output:

```
(2, 3)
```


## **2.4 The `ndim` Attribute**

Shows the number of dimensions.

```python
print(a.ndim)
```

### Output:

```
2
```

## **2.5 The `dtype` Attribute**

Shows the data type of elements.

```python
print(a.dtype)
```

### Output:

```
int64
```

## **2.6 The `size` Attribute**

Returns number of elements.

```python
print(a.size)
```

### Output:

```
6
```

## **2.7 Creating an Array with a Custom Data Type**

```python
a = np.array([1, 2, 3], dtype=np.int32)
print(a, a.dtype)
```

### Output:

```
[1 2 3] int32
```

## **2.8 Using the `type()` Function**

```python
a = np.array([1,2,3])
print(type(a))
```

### Output:

```
<class 'numpy.ndarray'>
```

# **3. Creating Arrays with Ones and Zeros (Default float64)**


## **3.1 Array of Ones — `np.ones()`**

```python
ones_array = np.ones((2,3))
print(ones_array)
print(ones_array.dtype)
```

### Output:

```
[[1. 1. 1.]
 [1. 1. 1.]]
float64
```

**With a Custom Data Type**

```python
ones_array = np.ones((2,3), dtype=np.int64)
print(ones_array)
print(ones_array.dtype)
```

### Output:

```
[[1 1 1]
 [1 1 1]]
int64
```

## **3.2 Ones with Shape and Data Type of Another Array — `ones_like()`**

```python
a = np.array([[5.,6.,7.],[8.,9.,10.]])
b = np.ones_like(a)
print(b)
print(b.dtype)
```

### Output:

```
[[1. 1. 1.]
 [1. 1. 1.]]
float64
```

**Another Example**

```python
a = np.array([[5,6,7],[8,9,10]])
b = np.ones_like(a)
print(b)
print(b.dtype)
```

### Output:

```
[[1 1 1]
 [1 1 1]]
int64
```

## **3.3 Array of Zeros (Default Data Type: float64) — `np.zeros()`**

```python
zeros_array = np.zeros((2,3))
print(zeros_array)
print(zeros_array.dtype)
```

### Output:

```
[[0. 0. 0.]
 [0. 0. 0.]]
float64
```

**With a Custom Data Type**

```python
zeros_array = np.zeros((2,3), dtype=np.int32)
print(zeros_array)
print(zeros_array.dtype)
```

### Output:

```
[[0 0 0]
 [0 0 0]]
int32
```


## **3.4 Zeros with Shape and Data Type of Another Array — `zeros_like()`**

```python
a = np.array([[3,4],[5,6]])
z = np.zeros_like(a)
print(z)
print(z.dtype)
```

### Output:

```
[[0 0]
 [0 0]]
int64
```

**Another Example**

```python
a = np.array([[3,4.0],[5,6]])
z = np.zeros_like(a)
print(z)
print(z.dtype)
```

### Output:

```
[[0. 0.]
 [0. 0.]]
float64
```

# **4. Creating Numerical Sequences**


## **4.1 The `arange()` Function**

Creates a sequence from start → stop with step.

```python
vec = np.arange(0, 10, 2)
print(vec)
print(vec.dtype)
```

### Output:

```
[0 2 4 6 8]
int64
```

## **4.2 The `linspace()` Function**

Creates `n` evenly spaced numbers between start and stop.

```python
vec = np.linspace(0, 1, 10)
print(vec)
```

### Output:

```
[0.         0.11111111 0.22222222 0.33333333 0.44444444
 0.55555556 0.66666667 0.77777778 0.88888889 1.        ]
```

## **5. A Summary Table**


| **Category**            | **Method / Attribute**          | **Description / Purpose**                                         | **Example / Output**                                  |
| ----------------------- | ------------------------------- | ----------------------------------------------------------------- | ----------------------------------------------------- |
| **Array Creation**      | `np.array()`                    | Creates a NumPy array from a Python list or other sequence        | `np.array([1,2,3])` → `[1 2 3]`                       |
|                         | `dtype` (attribute)             | Specifies or shows the data type of array elements                | `np.array([1,2,3], dtype=np.int32)` → `[1 2 3] int32` |
|                         | `type()`                        | Returns the type of the object (NumPy arrays are `numpy.ndarray`) | `type(a)` → `<class 'numpy.ndarray'>`                 |
| **Array Properties**    | `shape`                         | Returns array dimensions (rows, columns, etc.)                    | `a.shape` → `(2,3)`                                   |
|                         | `ndim`                          | Returns number of dimensions                                      | `a.ndim` → `2`                                        |
|                         | `size`                          | Returns total number of elements                                  | `a.size` → `6`                                        |
|                         | `dtype`                         | Returns data type of elements                                     | `a.dtype` → `int64`                                   |
| **Special Arrays**      | `np.ones(shape, dtype=...)`     | Creates an array filled with 1s                                   | `np.ones((2,3))` → `[[1. 1. 1.],[1. 1. 1.]]`          |
|                         | `np.zeros(shape, dtype=...)`    | Creates an array filled with 0s                                   | `np.zeros((2,3))` → `[[0. 0. 0.],[0. 0. 0.]]`         |
|                         | `np.ones_like(array)`           | Creates an array of 1s with same shape and dtype as another array | `np.ones_like(a)` → `[[1. 1. 1.],[1. 1. 1.]]`         |
|                         | `np.zeros_like(array)`          | Creates an array of 0s with same shape and dtype as another array | `np.zeros_like(a)` → `[[0 0 0],[0 0 0]]`              |
| **Numerical Sequences** | `np.arange(start, stop, step)`  | Creates a sequence of numbers with a fixed step                   | `np.arange(0,10,2)` → `[0 2 4 6 8]`                   |
|                         | `np.linspace(start, stop, num)` | Creates `num` evenly spaced numbers between start and stop        | `np.linspace(0,1,10)` → `[0. 0.11111111 ... 1.]`      |
