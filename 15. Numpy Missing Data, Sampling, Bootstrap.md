# **1. Missing Data (NumPy)**

NumPy provides tools to **identify missing data**, **sample data**, and **create bootstrap samples** for statistical analysis and machine learning preprocessing.


## **1.1 Identifying Missing Data**

In NumPy, **missing values** are represented as `NaN` (Not a Number). Use `np.isnan()` to detect them.

```python
import numpy as np

data = np.array([1, 2, np.nan, 4, np.nan])
print("Original array:", data)

# Identify missing values
missing_mask = np.isnan(data)
print("Missing values mask:", missing_mask)
```

**Output:**

```
Original array: [ 1.  2. nan  4. nan]
Missing values mask: [False False  True False  True]
```

## **1.2 Removing Rows with Missing Data (2-D Arrays)**

For 2-D arrays, you can remove rows that contain any `NaN` using boolean indexing.

```python
data2D = np.array([[1, 2, 3],
                   [4, np.nan, 6],
                   [7, 8, 9]])

# Remove rows containing NaN
clean_data = data2D[~np.any(np.isnan(data2D), axis=1)]
print("2-D array with rows containing NaN removed:\n", clean_data)
```

**Output:**

```
2-D array with rows containing NaN removed:
[[1. 2. 3.]
 [7. 8. 9.]]
```

**> `~` represents logical NOT, `np.any(..., axis=1)` checks if any element in a row is NaN.**


# **2. Sampling**
Use `np.random.choice()` to sample elements randomly from an array with/without replacement.

## **2.1 Random Sampling Without Replacement — `np.random.choice()`**

Data must be `1-dimensional`.

### Example: Selects 3 random elements from a 1-D array **without replacement**.

```python
data = np.array([10,20,30,40,50])
sample = np.random.choice(data, size=3, replace=False)
print(sample)
```

**Output (example):**

```
[40 10 20]
```

## **2.2 Random Sampling With Replacement— `np.random.choice()`**

### Example: Selects 3 random elements from a 1-D array **with replacement**.

```python
data = np.array([10,20,30,40,50])
sample = np.random.choice(data, size=3, replace=True)
print(sample)
```

**Output (example):**

```
[40 40 10]
```

# **3. Bootstrap**
Bootstrap sampling allows sampling with/without replacement, so the same element with replacement approach can appear multiple times. Useful for **generating multiple datasets** for simulations or model training.

## 3.1 **Random sampling with replacement**

**Example**

```python
data = np.array([1,2,3,4,5])
bootstrap_samples = np.random.choice(data, size=(3,5), replace=True)
print(bootstrap_samples)
```

**Output**

```
[[1 4 3 5 2]
 [2 3 4 4 1]
 [5 2 1 3 5]]
```

## 3.2 **Random sampling with replacement**

Number of **elements in Data and elements in wanted array should be equal**: for example if data has 6 elements, then and array should be in the shape of: `(3, 2), (2, 3), (1, 6), or (6, 1)`.

**Example**

```python
data = np.array([1,2,3,4,5,6])
bootstrap_samples = np.random.choice(data, size=(3,2), replace=False)
print(bootstrap_samples)
```

**Output**

```
[[5 1]
 [3 2]
 [6 4]]
```
