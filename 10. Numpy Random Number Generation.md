
# **1. Random Number Generation**

NumPy provides a variety of functions to generate random numbers and arrays. Random numbers are commonly used in simulations, testing, sampling, and machine learning.

Before using random number functions, make sure NumPy is imported:

```python
import numpy as np
```

## **1.1 Random Integers — `np.random.randint()`**

Generates random integers from a given range `[low, high)`.

**Example: Single random integer between 0 and 100**

```python
rand_int = np.random.randint(0, 100)
print(rand_int)
```

**Output**

```
42
```

**Example: Random array of integers between 0 and 10 with shape (3,5)**

```python
rand_array = np.random.randint(0, 10, size=(3,5))
print(rand_array)
print(rand_array.dtype)
```

**Output (example):**

```
[[9 0 4 0 8]
 [1 3 4 4 6]
 [6 1 0 0 1]]
int32
```

## **1.2 Random Floats Between 0 and 1 — `np.random.random()`**


**Example: Generates a random float number in the interval `[0,1)`.**

```python
rand_float_num = np.random.random()
print(rand_float_num)
```

**Output**

```
0.2075686697888982
```

**Example: Generates random floats array in the interval `[0,1)`.**

**Dimensions as a tuple.**

```python
rand_float_array = np.random.random((5,3))
print(rand_float_array)
```

**Output**

```
[[0.9493572  0.1567164  0.9957091 ]
 [0.50639052 0.394163   0.34063729]
 [0.59381416 0.42980552 0.23225385]
 [0.03363283 0.33901624 0.58120312]
 [0.90551102 0.93485157 0.4086409 ]]
```

## **1.3 Uniform Distribution — `np.random.rand()`**

**A uniform distribution means: **Every value in a range has the** same chance of happening**.
There is **no bias toward any number**.

**Uniform** = Equal chance for everything in the interval.

Generates random floats in `[0,1)` with **uniform distribution**. You can specify dimensions.

**Example: Generates a random number with uniform distribution**

```python
uniform_number = np.random.rand()
print(uniform_number)
```

**Output**

```
0.5600022298428707
```

**Example: Generates an array with uniform distribution**

**Dimensions as separate arguments (not a tuple).**

```python
uniform_array = np.random.rand(4,2)
print(uniform_array)
```

**Output**

```
[[0.417022   0.72032449]
 [0.00011437 0.30233257]
 [0.14675589 0.09233859]
 [0.18626021 0.34556073]]
```

### Difference between np.random.random() and np.random.rand()

| Feature      | `np.random.random()`                                      | `np.random.rand()`                                            |
| ------------ | --------------------------------------------------------- | ------------------------------------------------------------- |
| Input format | **Tuple** specifying shape `(rows, cols, ...)`            | **Separate arguments** for each dimension `d0, d1, ...`       |
| Example      | `np.random.random((2,3))`                                 | `np.random.rand(2,3)`                                         |
| Output       | Array of floats `[0.0, 1.0)`                              | Array of floats `[0.0, 1.0)`                                  |
| Flexibility  | Can accept a single integer (`size=5`) or a tuple `(2,3)` | Only separate dimension arguments, cannot pass a single tuple |



## **1.4 Normal (Gaussian) Distribution — `np.random.normal()`**

Generates random numbers following a normal (Gaussian) distribution. You can set the mean, standard deviation, and size (the famous **bell-shaped curve**).

**What is a Normal Distribution?**
* Most values are near the average (center of the bell).
* Fewer values appear far from the average.
* Data is **symmetrical around the mean**.

**Parameters**
* `loc` → the mean of the distribution
* `scale` → the standard deviation (spread or width of the bell curve)
* `size` → how many numbers (or what shape) to generate
* e.g: `np.random.normal(loc=0, scale=1, size=5)`

**Explanation**
* The numbers are **centered around 0** (because loc=0).
* **Most values are close to 0.**
* Some values are bigger or smaller, depending on the spread (**scale=1 means standard deviation = 1**).

**Example: Generates a number**

```python
# Mean = 0, Std Dev = 1, 1 numbers
normal_number = np.random.normal(0, 1)
print(normal_number)
```

**Output**

```
-0.24757258590219688
```


**Example 2: a vector with 5 numbers**

```python
# Mean = 0, Std Dev = 1, 5 numbers
normal_vector = np.random.normal(0, 1, 5)
print(normal_vector)
```

**Output**

```
[-0.33427783 -0.56687775  0.14498332 -1.30972017 -0.13057718]
```

**Example 3: an array with shape=(4,3)**

```python
# Mean = 0, Std Dev = 1, 5 numbers
normal_array = np.random.normal(0, 1, (4,3))
print(normal_array)
```

**Output**

```
[[-0.66378235 -0.00302825  0.02953163]
 [-0.01306537 -0.58374742 -0.55927953]
 [-0.8645971   0.75480748 -1.0245291 ]
 [-0.54025888  0.3107023  -0.35843017]]
```

## **1.5 Binomial Distribution (Multiple Repeated Bernoulli Trials) — `np.random.binomial()`**

Generates numbers according to a binomial distribution: number of successes in `n_trials` with a given probability.

**Example: Generates a vector**
Parameters:
* `n = 10` → number of trials
* `p = 0.5` → probability of success in each trial
* `size = 5` → how many random binomial values to generate

So this generates 5 random numbers, where each number is the **count of successes in 10 trials** when **success chance = 50%**.

**Each value will be between 0 and 10.**

```python
binomial_vector = np.random.binomial(10, 0.5, 5)
print(binomial_vector)
```

**Output (example):**

```
[6 4 5 7 3]
```

Explanation of output:
* **First number 6**: In 10 trials, **success happened 6 times**
* **Second number 4**: In 10 trials, **success happened 4 times**
* And so on …

**Every time you run the code, you will get different random results.**


**Example: Generates an array**

```python
# 10 trials, probability 0.5, shape=(4, 3)
binomial_array = np.random.binomial(10, 0.5, (4, 3))
print(binomial_array)
```

**Output**

```
[[3 5 4]
 [7 3 5]
 [8 5 6]
 [7 6 5]]
```

### **Bernoulli Distribution**
A **single experiment** with two possible outcomes (Success (1) and Failure (0)). **Bernoulli is just a binomial with `n = 1`**.

**Bernoulli Example**:
* Flipping a coin once.

**Binomial Example**:
* Flipping a coin for **e.g. 10 times** and counting how many times it lands heads.

## **1.6 Poisson Distribution — `np.random.poisson()`**
Poisson distribution is used for random events happening independently, like:
* Number of emails you receive per hour
* Number of cars passing a road per minute
* Number of phone calls in a call center per minute
  
`np.random.poisson(λ, size)` generates `size` random numbers from a Poisson distribution.

For example: `np.random.poisson(3, 5)`:
* `lam = 3` → the expected rate (“On average, 3 events happen.”)
* `size = 5` → generate 5 random Poisson values (The code simulates this 5 times.)
* Each result is a random number **around 3** (sometimes lower, sometimes higher).


**Example 1:**
```python
poisson_array = np.random.poisson(3, 5)
print(poisson_array)
```

**Output**

```
[3 2 5 3 1]
```

**Example 2:**

```python
poisson_array = np.random.poisson(3, (3, 6))
print(poisson_array)
```

**Output**

```
[[3 2 1 6 2 2]
 [2 2 2 6 1 1]
 [2 6 3 2 3 4]]
```


## **1.7 Exponential Distribution — `np.random.exponential()`**

Generates numbers following an exponential distribution with a given scale parameter. The exponential distribution is used to model **waiting times** between random events.

**Real-Life Examples of Exponential Distribution**
* Time until a bus arrives
* Time between phone calls in a call center
* Time until the next customer enters a shop

**Synatx and Parameters**:
* `np.random.exponential(scale, size)`
* **scale**: the average waiting time (mean)
* **size**: how many random values you want

**Explanation**: Think of it like this:
* Events happen randomly.
* You want to know **how long you wait until the next event**.
* `scale = 2` means you usually **wait about 2 units of time**.
* Every result is different because it’s random

**Example 1**

```python
exponential_array = np.random.exponential(1.0, 5)
print(exponential_array)
```

**Output**

```
[0.60148936 1.10114338 0.15503535 0.0706073  1.80188218]
```

**Example 2**

```python
exponential_array = np.random.exponential(1.0, (3, 5))
print(exponential_array)
```

**Output**

```
[[0.45879615 2.07640384 2.93760958 2.07313628 1.44088804]
 [1.52768134 0.1421137  0.13873268 0.06548415 1.89742853]
 [1.1328702  0.04669695 0.96139881 0.5397564  1.78689839]]
```

## **1.8 Log-normal Distribution — `np.random.lognormal()`**

Generates numbers following a log-normal distribution **with given mean and standard deviation** of the underlying normal distribution.

If a value **comes from a normal (Gaussian) distribution after taking the log**, then the original value follows a log-normal distribution.
* The numbers are **always positive**.
* They are usually **skewed to the right** (many small values, few very large ones).

**Parameters**
* **mean**: mean of the underlying normal distribution
* **sigma**: **standard deviation** of the underlying normal distribution
* **size**: shape (**how many numbers** to generate)

**Example 1**

```python
lognormal_array = np.random.lognormal(0, 1, 5)
print(lognormal_array)
```

**Output**

```
[0.61416215 0.40070732 2.50664825 0.40887314 1.97873849]
```

**Example 2**

```python
lognormal_array = np.random.lognormal(0, 1, (3, 4))
print(lognormal_array)
```

**Output**

```
[[0.51888889 0.32159786 2.87287689 2.68750022]
 [0.74517665 0.70773685 3.84793441 0.74136347]
 [0.08892749 0.5999199  1.54462875 3.63327694]]
```

# **2. Summary Table: Random Number Generation**

| **Function**                             | **Purpose / Description**                                   |
| ---------------------------------------- | ----------------------------------------------------------- |
| `np.random.randint(low, high, size)`     | Random integers in `[low, high)`.                           |
| `np.random.random(size)`                 | Random floats in `[0,1)`.                                   |
| `np.random.rand(d1,d2,...)`              | Uniform random floats in `[0,1)` with given shape.          |
| `np.random.normal(mean, std, size)`      | Normal (Gaussian) random numbers.                           |
| `np.random.binomial(n_trials, p, size)`  | Binomial distribution (number of successes).                |
| `np.random.poisson(lam, size)`           | Poisson distribution.                                       |
| `np.random.exponential(scale, size)`     | Exponential distribution.                                   |
| `np.random.lognormal(mean, sigma, size)` | Log-normal distribution.                                    |
| `np.random.choice(array, size, replace)` | Random sampling from an array, with or without replacement. |
