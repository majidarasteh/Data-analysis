# **1. Image Loading & Data Analysis (NumPy)**

NumPy can handle images as **arrays**. Each image is represented as a **3-D array** (height × width × channels). NumPy, combined with libraries like **Matplotlib**, makes it easy to **load, analyze, and manipulate images**.

---

## **1.1 Loading an Image with Matplotlib**

To load an image, we use `matplotlib.pyplot.imread()`.

```python
import numpy as np
import matplotlib.pyplot as plt

# Load an image
img = plt.imread("path_to_image.jpg")  # Replace with your image path

# Display the image
plt.imshow(img)
plt.axis('off')  # Remove axes
plt.show()

# Check type and shape
print("Type:", type(img))
print("Shape:", img.shape)
```

**Output (example):**

```
Type: <class 'numpy.ndarray'>
Shape: (480, 640, 3)
```

> Explanation:
>
> * `(480, 640, 3)` → 480 pixels height, 640 pixels width, 3 color channels (RGB).


## **1.2 Accessing Image Pixels**

Once loaded, an image behaves like a NumPy array. You can **access pixels and channels** using indexing.

```python
# Access the top-left pixel (row 0, column 0)
pixel = img[0, 0]
print("Top-left pixel value:", pixel)

# Access the red channel of the pixel
red_value = img[0, 0, 0]
print("Red channel of top-left pixel:", red_value)
```

**Output (example):**

```
Top-left pixel value: [255 128 64]
Red channel of top-left pixel: 255
```


## **1.3 Image Shape and Dimensions**

```python
print("Image shape:", img.shape)    # Height x Width x Channels
print("Number of dimensions:", img.ndim)
print("Total number of pixels:", img.size)
```

**Output (example):**

```
Image shape: (480, 640, 3)
Number of dimensions: 3
Total number of pixels: 921600
```

## **1.4 Converting Image to Grayscale**

Convert RGB images to grayscale by averaging the color channels.

```python
# Convert to grayscale
gray_img = np.mean(img, axis=2)
print("Grayscale image shape:", gray_img.shape)

# Display grayscale image
plt.imshow(gray_img, cmap='gray')
plt.axis('off')
plt.show()
```

**Output (example):**

```
Grayscale image shape: (480, 640)
```

## **1.5 Image Statistics**

NumPy allows simple **image analysis** with aggregation functions:

```python
print("Max pixel value:", np.max(img))
print("Min pixel value:", np.min(img))
print("Mean pixel value:", np.mean(img))
print("Standard deviation:", np.std(img))
```

**Output (example):**

```
Max pixel value: 255
Min pixel value: 0
Mean pixel value: 128.5
Standard deviation: 64.2
```

> These stats are useful for normalization, contrast adjustment, and understanding image intensity distribution.


## **1.6 Image Normalization**

Normalize pixel values to the range `[0, 1]` for machine learning.

```python
normalized_img = img / 255.0
print("Max after normalization:", np.max(normalized_img))
print("Min after normalization:", np.min(normalized_img))
```

**Output:**

```
Max after normalization: 1.0
Min after normalization: 0.0
```

## **1.7 Image Slicing and Cropping**

You can **slice images** just like NumPy arrays to crop or extract regions.

```python
# Crop the top-left 100x100 pixels
cropped_img = img[:100, :100, :]
plt.imshow(cropped_img)
plt.axis('off')
plt.show()
```

## **1.8 Flattening an Image**

Flatten a 2-D or 3-D image array into a 1-D vector for analysis or feeding into ML models.

```python
flat_img = img.flatten()
print("Flattened image shape:", flat_img.shape)
```

**Output (example):**

```
Flattened image shape: (921600,)
```
